# <font color=#1e90ff> Knižnica pyb.I2C </font>

I2C rozhranie je možné v MicroPython-e používať v natívnom (hardwarovom) mode viazanom piny určené hardwarovou konfiguráciou ako aj v emulovanom softwarovom móde. 

```C
#define MICROPY_HW_I2C1_SCL         (pin_B8)
#define MICROPY_HW_I2C1_SDA         (pin_B9)

#define MICROPY_HW_I2C2_SCL         (pin_B10)
#define MICROPY_HW_I2C2_SDA         (pin_B11)

#define MICROPY_HW_I2C3_SCL         (pin_C0)
#define MICROPY_HW_I2C3_SDA         (pin_C1)
```
    
## <font color=#ff4f00>  Inicializácia I2C rozhrania </font>

Obsluha I2C rozhrania je implementovaná v module *pyb*. V mcu STM32L476 sú piny I2C premapované, konfuguráciu pinov SDA/SCL získame výpisom vlastností rozhrania 

```Python
from machine import I2C
i2c = I2C(1,freq=400000)       # inicializacia HW SPI(1)
print(i2c)                     # konfiguracia I2C(1, scl=B8, sda=B9 ...)
print(i2c.scan())               # scan zbernice, [ ... ]
```

## <font color=#ff4f00> Inicializácia SW rozhrania </font>

Pri inicializácii je potrebné definovať piny portov, ku ktorým budú pripojené periférie.

```Python
from machine import I2C
from machine import SoftI2C 

sic = SoftI2C(sda='PB7', scl='B6', freq=400000)
print(sic.scan())                # -> [ ... ]
